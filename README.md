# Task Manager (JavaScript)

A simple task manager built with vanilla JavaScript, HTML, and CSS. Users can add, delete, and mark tasks as completed.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Data persists using `localStorage`
